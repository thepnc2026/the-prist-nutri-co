# The Prist Nutri Co. - Premium Nutrition E-Commerce Website

## Overview

The Prist Nutri Co. is a modern, premium e-commerce website for a nutrition brand. Built with vanilla HTML5, CSS3, and JavaScript, featuring glassmorphism design, smooth animations, and a fully responsive mobile-first layout.

## Features

✨ **Modern Design**
- Glassmorphism UI with soft shadows
- Premium typography and green & white branding
- Smooth animations and transitions
- Fully responsive design (mobile-first)
- Dark/Light mode toggle

🛍️ **E-Commerce Features**
- Product search and category filtering
- Shopping cart with quantity updates
- Wishlist functionality
- Product details with reviews and ratings
- WhatsApp checkout integration
- One-page checkout

🔧 **Performance & SEO**
- Lazy loading for images
- Smooth scroll behavior
- Loading animations
- Meta tags and Open Graph
- Twitter Cards
- Structured data (Schema.org)
- sitemap.xml and robots.txt

♿ **Accessibility**
- WCAG 2.1 AA compliant
- Semantic HTML
- ARIA labels and roles
- Keyboard navigation support
- High contrast ratios

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Flexbox, Grid, animations
- **JavaScript (ES6)**: Vanilla JS (no frameworks)
- **JSON**: Product database

## Installation

1. Clone the repository:
```bash
git clone https://github.com/thepnc2026/the-prist-nutri-co.git
cd the-prist-nutri-co
```

2. Open in a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server

# Or use VS Code Live Server extension
```

3. Visit `http://localhost:8000`
